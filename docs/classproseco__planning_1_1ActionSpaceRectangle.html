Gets the action class of an action given the the vehicle and its action space.

The action space is divided into five areas corresponding to the five basic action classes.

  • : accelerate
  • : decelerate 0 : Keep constant velocity and lateral position R : Change lateral position to the right L : Change lateral position to the left
    Note
    The division of the action space is not tuned and solely uses the points of the trajectory and not the entire rectangle of the vehicle
    Parameters
    actionThe action to classify.
    vehicleThe vehicle.
    Returns
    ActionClass The action class of the action.

Determines whether an action results in a lane change.

A lane change can only occur if the lateral change is non-zero and larger than the absolute distance to either of the lanes.

Parameters

distanceToRightLaneThe distance to the right lane.
Returns

false If the action does not result in a lane change.
